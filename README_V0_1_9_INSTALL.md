# Avi Oren Aviation — Version 0.1.9

## Changes
Homepage lower four boxes now show:
1. LHKA
2. Current temperature in LHKA/Kalocsa area
3. Current pressure in mb/hPa
4. Current wind direction and speed

Weather is fetched client-side from Open-Meteo for the LHKA/Kalocsa area.
This is for quick situational awareness only. Use official aviation weather before flight.

## Install
Replace/upload the `/frontend` folder, commit, then Render → Manual Deploy.
