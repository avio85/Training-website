# Avi Oren Aviation — V3 Bright UI

Install:
1. Delete the current /frontend folder in GitHub.
2. Upload this new /frontend folder.
3. Commit.
4. Render → Manual Deploy → Deploy latest commit.

Changes:
- Bright aviation theme.
- Airfields are merged into Briefing Room.
- LHKA is Kalocsa and opens the provided VFR chart PDF.
- Schedule view filters: daily / by FI / by student.
- Weather module with public METAR/TAF quick fetch and HungaroMet link.
- Aircraft/G1000 teaching pages.
- Modular CPL roadmap and requirements overview.
